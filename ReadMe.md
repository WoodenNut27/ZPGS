# ZPGS — 雑にPhysBoneのGrab設定を変更するやつ

概要
- Unity/VRChat用の補助ツール。PhysBone（VRChatの物理ボーン）に対して一括でGrab設定を適用します。

要件
- Unity 2022.3.22f1 (他のバージョンでも動くかもしれませんが、テストしていません。ご了承ください)
- VRChat SDK 3.10.1 (他のバージョンでも動くかもしれませんが、テストしていません。ご了承ください)

インストール
- 「ZGPS-ver x.x.x.unitypackage」をインポートしてください。(xはZGPSのバージョン)

使い方（簡易）
1. ツールバーの「Tools/ZGPS」からウィンドウを表示
2. 対象オブジェクトにアバターやPhsyBoneコンポーネントを含むゲームオブジェクトをドラッグアンドドロップする
3. PhsyBoneのフルパスがリストに表示されるので、変更したいオブジェクトにチェックを入れる
4. リスト下のGrab設定、Pose設定を変更する
