# ZPGS — 雑にPhysBoneのGrab設定を変更するやつ

概要
- Unity / VRChat 用の補助ツール。PhysBone ( VRChat 専用のボーン) に対して一括で Grab / Pose 設定を適用します。

要件
1. Unity バージョン
    - Unity 2022.3.22f1
    
    (他のバージョンでも動くかもしれませんが、テストしていません。ご了承ください)

2. SDK バージョン
    - VRChat SDK 3.10.1
    - VRChat SDK 3.10.2
    
    (他のバージョンでも動くかもしれませんが、テストしていません。ご了承ください)

インストール
- 「ZGPS-ver x.x.x.unitypackage」をインポートしてください。(xはZGPSのバージョン)

使い方（簡易）
1. ツールバーの「Tools/ZGPS」からウィンドウを表示
2. 対象オブジェクトにアバターや PhsyBone コンポーネントを含むゲームオブジェクトをドラッグアンドドロップする
3. PhsyBone のフルパスがリストに表示されるので、変更したいオブジェクトにチェックを入れる
4. リスト下の Grab 設定、 Pose 設定を変更する

ライセンス
- NYSL
